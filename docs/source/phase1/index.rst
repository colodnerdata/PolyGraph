Phase 1: Generators
===================

Phase 1 focuses on foundational combinatorial polyhedron generators used by
higher layers (algorithms, geometry, visualization, and export).

Implemented sections
--------------------

- ``polygraph.generators.prisms``: prism and antiprism parametric families.
- ``polygraph.generators.platonic``: the five Platonic solids.
- ``polygraph.generators.johnson``: pyramid and dipyramid families (plus
  explicit deferred Johnson stubs).
