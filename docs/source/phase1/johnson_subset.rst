Phase 1c: Johnson Families (Subset)
===================================

Module: ``polygraph.generators.johnson``

Phase 1 includes the simplest parametric Johnson families:

- ``pyramid(n)``
- ``dipyramid(n)``
- ``bipyramid(n)`` (alias)

The module also documents deferred Johnson-solid constructors that currently
raise ``NotImplementedError``.
