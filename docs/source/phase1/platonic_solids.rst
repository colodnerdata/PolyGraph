Phase 1b: Platonic Solids
=========================

Module: ``polygraph.generators.platonic``

This module contains combinatorial generators for the five Platonic solids:

- ``tetrahedron()``
- ``cube()``
- ``octahedron()``
- ``dodecahedron()``
- ``icosahedron()``

Each generator returns a closed ``DartMap`` with consistently oriented faces.
